## Quick Start

```bash
# 1. Install Ollama
# https://ollama.com/download

# 2. Pull a model
ollama pull llama3.2

# 3. Clone this repo
git clone https://github.com/SenpaiCore/Senpai-Ollama-Tools.git
cd Senpai-Ollama-Tools

# 4. Install dependencies
pip install -r requirements.txt

# 5. Run any agent
python agents/proposal_generator.py
