  <h3>What is Senpai-Ollama-Tools?</h3>
  <p>
    A carefully crafted collection of <strong>ready-to-use local AI agents</strong> powered by Ollama. 
    Built with Python and LangChain, these agents run 100% offline on your laptop or desktop — no internet, no subscriptions, no data leaving your machine.
  </p>
  <p>
    Designed specifically for Indian freelancers, students, and developers who want premium AI capabilities without the recurring costs or privacy concerns of cloud services.
  </p>

  <pre><code># Step 1: Install Ollama (One-time only)
# Visit → https://ollama.com/download and install for your OS

# Step 2: Download a fast model
ollama pull llama3.2

# Step 3: Clone the repository
git clone https://github.com/SenpaiCore/Senpai-Ollama-Tools.git
cd Senpai-Ollama-Tools

# Step 4: Install Python dependencies
pip install -r requirements.txt

# Step 5: Run the demo
python examples/example_usage.py
